== Description ==

Combine the power of Give WP and Zero BS CRM and automatically track donors and donations in Zero BS CRM.

<blockquote>An essential plugin for all Give users: Connect GiveWP to ZBS CRM</blockquote>

###Get More Donations via Give:

1. **Incredibly simple:** Install and forget automation
2. **Fully Leverage:** Your donor data in Zero BS CRM's management tools
3. **Free:** This extension is free for all Give and Zero BS CRM users


###Why add a CRM to Give?
Give is a useful donation platform as it is. Zero BS CRM is the no-nonsense CRM for WordPress. Keep your Donors happy by utilising Zero BS CRM's management tools. Make Notes. Send Follow Ups. Tag and Segment your donors. Supercharge this process with [ZBS Extensions](https://zerobscrm.com/extensions/) and [Give Addons](https://givewp.com/addons/).

Never miss a donation again by capturing your donations into Zero BS CRM. Manage your donors just like you'd manage your customers. Follow up with your Donors by utilising Zero BS CRM's management tools to keep your donors happy.

= Utilise all the features of Zero BS CRM =
You are running Give and want to power up your management of the people who donate to your cause. Enter Zero BS CRM. Through using Give to ZBS you can capture your Donors into your ZBS CRM.

= Automatically Capture: =
* Donor information
* First Name
* Last Name
* Email
* Donor History

Use the information with your CRM since Donors are captured as Contacts to your CRM. You can then use the wider features of Zero BS CRM such as

* View your Donations Sales in the Sales Dashboard. ([See a tour of the Sales Dashboard](https://www.youtube.com/watch?v=ju2aeLkrxcM))
* Email your Donors based on certain criteria
* Tag your Donors and quickly filter them 


== Installation ==

= Requires: =
* WordPress 4.0 or greater
* PHP version 5.3 or greater
* MySQL version 5.0.15 or greater

= Installing Give ZBS Connector =
1. First make sure you've got WordPress installed somewhere. We like to use crm.yourdomain.com
2. Install your plugin as below (you probably know how!)
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Use the menu to go to the settings page: Zero BS CRM->Settings->Give to configure the plugin (this is recommended)

For more documentation, please see: [Integrating Give WP with ZBS CRM](https://zerobscrm.com/zbs-blog/integrations/how-to-integrate-zero-bs-crm-with-give-wp/) or our [Knowledge Base](https://zerobscrm.com/kb)

###Automatic Install From WordPress Dashboard

1. Login to your the admin panel
2. Navigate to Plugins -> Add New
3. Search **Give to Zero BS CRM Connector**
4. Click install and activate respectively.

###Manual Install From WordPress Dashboard

If your server is not connected to the Internet, then you can use this method-

1. Download the plugin by clicking on the orange button above. A ZIP file will be downloaded.
2. Login to your siteâ€™s admin panel and navigate to Plugins -> Add New -> Upload.
3. Click choose file, select the plugin file and click install

###Install Using FTP

If you are unable to use any of the methods due to internet connectivity and file permission issues, then you can use this method-

1. Download the plugin by clicking on the red button above.A ZIP file will be downloaded.
2. Unzip the file.
3. Launch your favorite FTP client. Such as FileZilla, FireFTP, CyberDuck etc. If you are a more advanced user, then you can use SSH too.
4. Upload the folder to wp-content/plugins/
5. Log in to your WordPress dashboard.
6. Navigate to Plugins -> Installed
7. Activate the plugin
